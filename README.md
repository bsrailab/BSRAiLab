# BSR AI Architecture

Tổng hợp và so sánh các kiến trúc AI, bao gồm Copilot Studio, Azure AI Foundry, và kiến trúc dự án tùy chỉnh.

## Cài đặt

1. Clone repo
2. `npm install`
3. `npm run dev` để chạy development server
4. `npm run build` để build cho production

## Triển khai trên GitHub Pages

1. Push code lên GitHub repo tên `BSRAiArchitecture`
2. `npm run build`
3. `npm run deploy`
4. Enable GitHub Pages trong repo settings, chọn branch `gh-pages`

## Cấu trúc

- `src/App.tsx`: Component chính với sections cho từng kiến trúc
- `src/App.css`: Styling cơ bản
- Placeholder cho hình ảnh và animation sẽ được thay thế khi có dữ liệu thực tế.

## Tiếp theo

Khi có hình ảnh kiến trúc, thay thế placeholders trong `App.tsx` và thêm animation cho flow diagrams sử dụng thư viện như Framer Motion hoặc D3.js.
